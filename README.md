## Descripción

## Requisitos
 - Node
 - yarn

# Instalación
 1. Clonar los archivos del github

 2. Ingresar a la carpeta

 3. instalar node_modules a través de yarn
 > yarnaaa

 4. Subir servidor local
 > yarn dev

 5. Ingresar a la url que aparezca en consola
